## 它是什么

Southside Music 是一个 Windows 桌面音乐播放器。登录网易云音乐账号后，即可搜索、浏览和播放全曲库歌曲。它拥有独立的音频引擎，提供响度均衡、变速播放、频谱可视化等专业功能，同时支持逐字歌词和桌面悬浮窗。

简单来说：它把你网易云账号里的歌，用一种更纯粹、更专注的方式播放出来。

---

## 功能一览

**播放**
- 网易云全曲库搜索与流媒体播放，VIP 用户可获取无损音质
- 响度均衡——所有歌曲自动调整到相同感知音量，切歌不再忽大忽小
- 0.1× 到 3.0× 变速播放，变速后人声不失真
- 可选的立体声增强效果，耳机听感更开阔
- 自动跳过结尾无声段落，无缝衔接下一首
- 预加载下一首，切换几乎无延迟

**视觉**
- 实时频谱动画，跟随音乐跳动
- 逐字歌词高亮，支持中英翻译
- 桌面歌词——不透明的置顶悬浮窗，做其他事也能跟唱
- 明暗主题自动跟随系统

**管理**
- 浏览、播放网易云歌单
- 将云端歌曲下载到本地收藏夹，断网也能听
- 收藏夹支持多文件夹分类
- 歌曲导出为完整音频文件，自动写入封面和歌曲信息

**其他**
- 检测 GitHub 新版本并自动更新

## 安装

从 [Releases](https://github.com/Adreno5/SouthsideMusic/releases) 下载 `SouthsideMusic_win64_setup.exe`，双击运行。安装器会创建开始菜单快捷方式，同时注册卸载入口。

> 首次启动时会自动检测 ffmpeg 是否已安装。若未安装，程序会引导你完成安装。

## 初次使用

### 1. 登录

打开软件后，在左侧的 **Session** 中登录网易云：

- **手机号登录** — 输入手机号，获取验证码后填入
- **二维码登录** — 用网易云音乐 App 扫描屏幕上的二维码

游客身份几乎无法正常使用——几乎所有歌曲都只能试听 30 秒，且无法获取完整歌词。

### 2. 搜索

点击标题栏的搜索框，输入歌名或歌手名，回车。结果直接显示在下方列表中。双击即可播放。

### 3. 浏览歌单

在左侧收藏夹区域可以看到你的网易云歌单。点击加载其中的歌曲，双击播放。云端歌曲可以右键选择 "Add to Local Folder"，下载到本地收藏夹中保存。

### 4. 管理本地收藏

点击左侧收藏夹区域的 "Add Folder" 按钮创建文件夹，用于分类管理歌曲。收藏到本地的歌曲即使不联网也能完整播放。

## 界面说明

### 搜索

点击顶部标题栏的搜索框，输入关键词回车即可。无需切换到特定页面，在任何界面都可以随时搜索。结果列表显示歌名、歌手和时长。双击播放，选中后可按"收藏"按钮保存到本地文件夹。

### 播放区

点击底部常驻的播放进度条，会展开完整的播放面板。布局分为左右两部分：

- **左侧** — 上方是专辑封面，下方是歌曲名称和歌手信息
- **右侧** — 滚动歌词，支持双语对照和逐字高亮

频谱动画和播放控制在底部常驻栏中，无需展开面板即可操作。

### 收藏夹

左侧栏的收藏夹区域集成了本地文件夹和云端歌单。点击 "Add Folder" 创建本地文件夹，点击云歌单即可加载歌曲。右键歌曲可以进行导出、添加到本地文件夹等操作。

### 设置

点击标题栏的设置图标进入。可调整的选项包括：

| 设置项 | 说明 |
|---|---|
| 响度目标 | 统一播放音量的基准值。数值越低音量越大，推荐保持 -16 |
| 播放速度 | 0.1× 到 3.0× |
| 播放模式 | 列表循环 / 单曲循环 / 随机 / 顺序播放 |
| 立体声增强 | 开启后声场更宽 |
| 跳过静音结尾 | 自动跳过末尾无声段落 |
| 静音阈值 | 判定"无声"的音量标准 |
| 频谱可视化 | 开关底部频率动画 |
| 桌面歌词 | 开关置顶歌词悬浮窗 |
| 输出设备 | 选择音频输出设备 |

## 高级功能

### 逐字歌词

当歌曲提供逐字时间数据时，播放面板会自动切换到逐字高亮模式——当前唱到的字以主题色高亮，已唱过的字渐变为灰色，未唱到的字保持正常颜色。翻译紧随其后，与逐字时间精确对齐。

### 桌面歌词

在设置中开启后，会弹出一个置顶悬浮窗实时显示歌词。窗口背景完全不透明，确保在任何背景下歌词都清晰可读。窗口可自由拖动到屏幕任意位置。

### 歌曲导出

右键已缓存的歌曲，选择 "Export" 即可导出为音频文件。导出时会自动写入封面图片、歌曲名称、歌手和专辑信息。

### 局域网广播

Southside Music 启动时会在本机 15489 端口启动一个广播服务。局域网内的其他设备可以连接获取实时播放状态、歌词内容和频谱数据，用于搭建歌词显示器、灯光控制器等扩展应用。

## 小贴士

- **空格键** 暂停 / 继续播放
- 在歌词区域**滚动鼠标滚轮**可以浏览歌词，并跳转到对应时间点
- 云歌单中的歌曲右键可以**添加到本地收藏夹**，方便离线收听
- 应用主题颜色会从当前播放歌曲的**专辑封面自动提取**
- 如果觉得音量整体不舒服，可以在设置中**调整响度均衡的目标值**
- 频谱动画的颜色会跟随应用主题颜色变化

## 开发

### 环境要求

Python 3.12+（推荐 3.12.7）、Windows 操作系统、[uv](https://docs.astral.sh/uv/) 包管理器、互联网连接（首次安装时需要）。

### 初次搭建

```bash
git clone https://github.com/Adreno5/SouthsideMusic.git
cd SouthsideMusic
python setup_workspace.py
```

`setup_workspace.py` 自动完成：安装 uv、下载 Python 3.12.7 嵌入式发行版、同步项目依赖、安装 Inno Setup 打包工具。

### 从源码运行

```bash
uv run src/main.py
```

### 类型检查

```bash
uv run mypy src/
```

### 构建

```bash
build.bat
```

`build.bat` 完成以下流程：

1. 使用 Nuitka 将 `launcher.py` 编译为独立可执行文件 `Launch.exe`
2. 将嵌入式 Python、源代码、图标、字体等汇总到 `build.result\raw\`
3. 生成应用图标
4. 调用 Inno Setup 编译安装器到 `build.result\installer\`

构建产物：

```
build.result\
├── raw\          运行目录（直接可运行）
└── installer\    安装器（SouthsideMusic_win64_setup.exe）
```

### 技术栈

| 层级 | 技术 |
|---|---|
| 界面 | PySide6 + PySide6-Fluent-Widgets |
| 音频 | pydub + sounddevice |
| 数学 | NumPy + SciPy（傅里叶变换、信号处理） |
| 元数据 | mutagen（ID3、Vorbis、MP4 标签） |
| API | pyncm（网易云音乐） |
| 网络广播 | Tornado |
| 打包 | Nuitka + Inno Setup |
| 字体 | HarmonyOS Sans SC |

### 配置

所有设置保存在 `config.json`。每秒检测一次是否被修改，有变动则自动保存。
